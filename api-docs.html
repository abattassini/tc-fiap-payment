üìã Available Endpoints

POST /v1/payment
Create a new payment for an order. Generates a QR code for payment via MercadoPago gateway. 
curl -X POST http://localhost:8082/v1/payment \
  -H "Content-Type: application/json" \
  -d '{
    "orderId": 123,
    "amount": 99.90
  }'
GET /v1/payment/{orderId}
Get payment details by order ID. Returns payment information including status, amount, and QR code. 
curl -X GET http://localhost:8082/v1/payment/123
GET /v1/payment/{orderId}/status
Get only the payment status for an order. Useful for polling payment status updates. 
curl -X GET http://localhost:8082/v1/payment/123/status
POST /payment/webhooks/notify
Webhook endpoint for MercadoPago payment notifications. Automatically updates payment status and notifies the order service. 
curl -X POST http://localhost:8082/payment/webhooks/notify \
  -H "Content-Type: application/json" \
  -d '{
    "id": "12345",
    "data": {
      "id": "payment_123"
    }
  }'
